POLÍTICA DE PRIVACIDADE

A presente Política de Privacidade contém informações sobre a coleta, uso, armazenamento, tratamento e proteção dos dados pessoais dos usuários e visitantes do site Encontre, com a finalidade de demonstrar absoluta transparência quanto ao assunto e esclarecer a todos interessados sobre os tipos de dados que são coletados, os motivos da coleta e a forma como os usuários podem gerenciar ou excluir as suas informações pessoais.

Esta Política de Privacidade aplica-se a todos os usuários e visitantes do site Encontre e integra os Termos e Condições Gerais de Uso do site Encontre.

O presente documento foi elaborado em conformidade com a Lei Geral de Proteção de Dados Pessoais (Lei 13.709/18). Ainda, o documento poderá ser atualizado em decorrência de eventual atualização normativa, razão pela qual se convida o usuário a consultar periodicamente esta seção.

SEÇÃO 1 - INFORMAÇÕES GERAIS

O site Encontre não coleta dados pessoais dos usuários e visitantes e não realiza transações financeiras. O site apenas envia informações sobre os produtos de locação e permite a seleção desses produtos. A comunicação sobre todos os produtos selecionados é realizada através da plataforma WhatsApp.

SEÇÃO 2 - COMO RECOLHEMOS OS DADOS PESSOAIS DO USUÁRIO E DO VISITANTE?

O site Encontre não coleta dados pessoais dos usuários e visitantes. A comunicação sobre os produtos de locação é realizada via WhatsApp, e o site não armazena dados pessoais dos usuários.

SEÇÃO 3 - QUAIS DADOS PESSOAIS RECOLHEMOS SOBRE O USUÁRIO E VISITANTE?

Como o site não coleta dados pessoais, não há dados pessoais armazenados ou utilizados.

SEÇÃO 4 - PARA QUE FINALIDADES UTILIZAMOS OS DADOS PESSOAIS DO USUÁRIO E VISITANTE?

Não há utilização de dados pessoais, pois o site não coleta essas informações.

SEÇÃO 5 - POR QUANTO TEMPO OS DADOS PESSOAIS FICAM ARMAZENADOS?

Como o site não coleta dados pessoais, não há armazenamento de dados.

SEÇÃO 6 - SEGURANÇA DOS DADOS PESSOAIS ARMAZENADOS

Como não há coleta de dados pessoais, não há medidas de segurança aplicáveis para dados pessoais.

SEÇÃO 7 - COMPARTILHAMENTO DOS DADOS

O site Encontre não compartilha dados pessoais, pois não coleta essas informações.

SEÇÃO 8 - COOKIES OU DADOS DE NAVEGAÇÃO

O site Encontre não utiliza cookies para coleta de dados de navegação.

SEÇÃO 9 - CONSENTIMENTO

Ao utilizar o site Encontre, o usuário está ciente de que não são coletados dados pessoais.

O usuário pode entrar em contato para dúvidas e solicitações através do e-mail: guilhermedevsistemas@gmail.com.

SEÇÃO 10 - ALTERAÇÕES PARA ESSA POLÍTICA DE PRIVACIDADE

Reservamos o direito de modificar essa Política de Privacidade a qualquer momento, então, é recomendável que o usuário e visitante revise-a com frequência.

SEÇÃO 11 - JURISDIÇÃO PARA RESOLUÇÃO DE CONFLITOS

Para a solução de controvérsias decorrentes do presente instrumento será aplicado integralmente o Direito brasileiro.

Os eventuais litígios deverão ser apresentados no foro da comarca em que se encontra a sede da empresa.
